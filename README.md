# ALGO Edge Streamlit App

Deploy with Streamlit Cloud using:

- Main file path: `app.py`
- Requirements file: `requirements.txt`

This build supports:
- Portfolio balance-history CSVs with `Date`, `NLV`, `Day_PL`, `Day_PL_Percent`, and `Deposits/Withdrawals`
- Individual strategy CSVs with date plus balance, return, or P/L columns
- Strategy selection when a `Strategy` column exists
- Combined P/L analysis for strategy files
- Sit-out overlays
- 1-year and 10-year Monte Carlo projections
